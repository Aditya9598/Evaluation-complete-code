What is this workspace?

This monorepo hosts four independent evaluation projects, each at a different skill tier. They do not share code, configuration, or business logic. The Examiner Hub lets you preview each project UI, check health status, and jump to API docs — all from one Railway deployment.

Projects at a glance

Project Folder Tier Stack Hub path (Railway)
Transaction Ledger transaction-ledger/ Basics (B1–B4) FastAPI + React /ledger/
Currency Converter currency-converter/ Intermediate (I1–I6) FastAPI + Node CLI + React /converter/
Fraud Score System Fraud-score-system/ Advanced (A1–A6) FastAPI + Node + Rust /fraud/ui/
Screen Scraper Ops screen-scraper/ Advanced (A1–A6) React + live API + local gateway /scraper/ops

How to use the hub

  1. Open the deployed URL — the hub loads at /.
  2. Click a project tab (Ledger, Converter, Fraud, Scraper) to preview its UI in the iframe.
  3. Use the toolbar links to open dashboards or API docs in a new tab.
  4. Check the green health dots — they confirm API and UI endpoints respond.
Rule for examiners: Treat each project folder as isolated. Do not share code, .env files, or business logic between them.

Basics & Intermediate eval tasks

These tiers test repo reading, API mapping, test discovery, greenfield FastAPI work, ER diagrams, flow traces, polyglot integration, Docker, and bug diagnosis.

Basics Transaction Ledger

Eval title: Repo reader and simple builder.
Project: FastAPI ledger with in-memory users, credit/debit transactions, balance tracking, React dashboard, and pytest suite.

B1 · 30 min

Repo artifact inventory

Find models, stores, configs, and frontend client code. Evidence in docs/eval/basics/B1-artifact-inventory.md.

B2 · 30 min

API endpoint map

Map every API route and React page. Evidence in docs/eval/basics/B2-api-endpoint-map.md. OpenAPI at /ledger/docs.

B3 · 15 min

Test discovery

Framework, test files, commands. Run pytest -v in transaction-ledger/.

B4 · 60 min

Greenfield FastAPI service

The ledger codebase itself is the deliverable: POST/GET transactions, GET balance, validation, ≥3 tests, README.

Intermediate Currency Converter

Eval title: Repo operator and polyglot builder.
Project: FastAPI /convert service, Node CLI client, React ER diagram UI, Docker, and bug-diagnosis branches.

I1 · 45 min

ER diagram from repo

Mermaid diagram with source citations. UI at /converter/er-diagram.

I2 · 45 min

End-to-end flow trace

Trace POST /convert through service layers with sequence diagram.

I3 · 60 min

Small safe change

Make a scoped change with tests in an unfamiliar repo.

I4 · 90 min

Polyglot pair

FastAPI service + Node CLI calling it. Two-terminal demo in README.

I5

Dockerize and run

Build and run via Dockerfile. Doc in docs/eval/intermediate/I5-docker.md.

I6

Bug diagnosis

Find and fix a seeded bug on branch eval/i6-seeded-bug.

Full task-to-file mapping lives in the repo root: EVAL_AND_PROJECTS_GUIDE.md and PROJECTS_DETAILED_GUIDE.txt.
